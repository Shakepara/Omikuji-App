# おみくじアプリ


## 概要
Javaで作成した絵文字入りのかわいいコンソールおみくじアプリです！
実行するたびにランダムな結果（大吉・中吉・小吉・大凶）と、ラッキーアドバイスが表示されます！


## 実行方法
1. このリポジトリをクローンまたはダウンロードします  
2. Java（JDK 11以上）がインストールされている環境で、`Main.java` を実行します
